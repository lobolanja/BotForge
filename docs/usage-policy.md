# BotForge Usage Policy

Version: 2026-05-08
Effective date: 2026-05-08
Privacy notice version: 2026-05-08

This draft is a practical product notice for beta use. It is not a substitute
for legal review.

## What This Bot Is

BotForge is a Telegram bot framework that receives user messages and returns
AI-assisted responses using the configured bot profile and model provider.

## AI Limitations

The bot can misunderstand requests, make mistakes, or produce incomplete
answers. It does not replace professional, legal, medical, financial, or safety
judgment.

## User Responsibilities

Do not send illegal content, credentials, secrets, or personal data that is not
needed for the conversation. You are responsible for checking important answers
before relying on them.

## Data We Process

BotForge may process Telegram user ids, messages, timestamps, bot interactions,
files if uploads are enabled, invite and login state, and technical logs needed
to run and secure the service.

## Conversation Memory

BotForge may keep recent conversation context or compacted memory when a profile
enables memory. Memory is used to provide more relevant replies.

## Analytics And Improvement

Analytics or training use is not silently enabled by accepting this required
usage policy. If analytics or training use is introduced, it should use separate
consent and separate version tracking.

## Files

If file uploads are enabled later, uploaded files may be processed to answer the
user's request and retained only as documented for that feature.

## Retention

Retention periods are configurable and should be documented before beta use with
real users. Operators should delete data that is no longer needed for service,
security, or audit purposes.

## User Rights And Contact

Users can stop using the bot by declining the policy or logging out. Requests
for access, correction, deletion, or consent withdrawal should be sent to the
project operator configured for the deployment.
